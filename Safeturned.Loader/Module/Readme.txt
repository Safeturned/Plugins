Safeturned Loader
----------------
Drop the `Safeturned.Loader` folder into your server’s `Modules` directory. Keep the folder whole (don’t just copy the files).  
Edit `config.json` to add your API key and adjust any settings.  
Then restart Unturned so it picks up your changes.  
Need more info? Visit http://github.com/safeturned/plugins or https://safeturned.com/.
