Safeturned Module
-----------------
Place the `Safeturned.Module` folder into `SafeturnedBuild` on your server (keep the folder intact).  
Open `config.json`, paste your API key, and adjust the scan settings if needed.  
Restart Unturned or reload the module so the new key takes effect.  
Want to learn more? Browse http://github.com/safeturned/plugins or https://safeturned.com/.
